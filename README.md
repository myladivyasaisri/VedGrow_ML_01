# VedGrow_ML_01 - Iris Flower Classification

This repository contains a beginner-level Machine Learning project to classify Iris flower species using Scikit-Learn.

## Project Requirements Met
- **EDA**: Visualized features using Seaborn pairplots.
- **Models Trained**: KNN, Decision Tree, and SVM.
- **Metrics Evaluated**: Accuracy, Precision, Recall, and F1-Score.
- **Visualizations**: Confusion matrix heatmaps for each model.
- **Custom Prediction**: Included a function to predict species for custom user inputs.

## How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Open the notebook in VS Code and run all cells.
